This is the code folder of my bachelor thesis, it contains the genetic
algorithm, simple_ga.py and the comparing algorithms, so the local search,
local_search.py, the greedy_algorithm, local_search_flexible.py and the random
search, random_search.py
For these algorithms all the needed functions and other code is included in this
folder as well.

The parent folder contains a part from this folder the results folder, which was
used to generate the data for the plots. (results)
It also contains the config file, which were used to configurate the
computations. (configfiles)

The INSTALL.txt contains some information concerning the installation, the
needed packages, ...

The EXAMPLES.txt contains some examples to run the code
